# CloudVero Consulting — Website Mockups

HTML/CSS mockups for the CloudVero Consulting website.

## Pages

| Page | File | Description |
|------|------|-------------|
| Homepage | `index.html` | Hero, problem cards, services overview, social proof, terminal demo, CTA |
| Services | `services.html` | Service breakdowns, tiered pricing cards, comparison table, FAQ |
| About | `about.html` | Origin story, founder bios, values, tech stack |
| Contact | `contact.html` | Contact form with service selector, info sidebar |
| Resources | `resources.html` | Blog/article template with category filters |

## Screenshots

Full-page screenshots are in `/screenshots` for sharing and review.

## Color Palette

Derived from the CloudVero logo:

- Steel Blue `#3da8d5` — primary accent, links, buttons
- Forge Orange `#f5a623` — highlights, pricing, category labels
- Dark Charcoal `#0c1117` / `#141c25` — backgrounds
- Silver Gray `#8a9bae` — secondary text

## Running Locally

Open any `.html` file directly in a browser — no build step required.

## Assets

Place the logo file at `assets/cloudverologo.PNG`.
